repo gồm 2 thư mục Lab và Code.
Lab--> chứa toàn bộ cấu trúc, thư viện inet từ omnet++.
Code--> chứa sources code của 3 bài: bai1, bai2, bai3.
Các sources code gồm 3 file cấu hình là .ned, .cc, ini.
Cách chạy các bài tập.
Biên dịch (Build):

    Trong OMNeT++ IDE, chọn Project → Clean.

    Chọn Project → Build Project (Hoặc Ctrl + B).

Chạy Mô phỏng (Run):

    Nhấp chuột phải vào omnetpp.ini → Run As → OMNeT++ Simulation.

    Trong cửa sổ TKENV mở ra, nhấn Run hoặc Step.
